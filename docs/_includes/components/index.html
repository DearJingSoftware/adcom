Index

Render a collection of items into a clear, compact list that supports common requirements like active states and metadata.

Examples

The main classes you need to apply are index to the container element, and index-item to each item in the collection. By itself, Adcom some very light adjustments to padding and margin to ready your list for a stacked list of items that can contain one or several rows of text.

Primary Index

This index is suited well for a main index page of objects. Display metadata about the item within index-item-meta. By including a <span class="state"> inside the item, you can also add a highlight bar that adapts to various states using the .active, .success, .warning, .info and .danger classes.
